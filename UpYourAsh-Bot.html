Road Map

Planned

Web Interface

The web interface will allow for a more dynamic use of the bot. Allowing the ability to use the bot through a visual interface rather than through commands. 

The web interface will allow for users and admins to manage roles, operation information, attendance and much more.

Certification Applications

This feature will help simplify applying for certifications.

It will do this will the user of a few things.

-It will only allow applications to certifications that are open for applications.

-It will only allow a user to apply to certifications that they qualify to apply for. E.g certifications that require a user to have a specific certifications before they apply.

-Allowing for the scheduling of operations to be done automatically upon approval and then notify both parties of the time and date.

Further details will be provided upon further development.

Training Scheduling

Allows for the scheduling of training for certifications to be easier to do.

As well as automatically notifying  those involved on the day as well as a certain time frame before it starts.

Polls & Votes

Polls and votes without having to manually end them, count votes and decided the outcome.

The bot will keep track of the votes as well as how much time is left of the poll/vote and once it ends it will output the results to a predefined place : D 

Further details will be provided upon further development.

Automatic Announcements

The bot will automatically operations, events, poll results as well as any predefined announcements with a set date and time to be announced.

Further details will be provided upon further development.

In Progress

User Profiles

User profiles are one of the core features of the bot. 
Each user within your server will automatically be created one.
They store information relating to levels, custom roles, economy and so much more. 

Other details will be provided upon further development.

Use Assigned Roles

Allows a user to assign certain roles to themselves through commands. Useful for roles relating to games etc.

Moderation 

The bot will be have integration with google calendar to allow for the scheduling of operations and certification training as well as any other 5Th column events. 

This will also play a key part in the automatic announcement of operations as well as messages those who said they are taking part when an operation is about to start.
